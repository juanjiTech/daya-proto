# daya-proto

## 快速开始

1. 安装软件
   - [buf](https://buf.build/docs/installation) 以及buf的IDE插件
   - `go install github.com/grpc-ecosystem/protoc-gen-grpc-gateway-ts`
   - `buf mod update`
2. 修改proto
3. 检查 `buf lint`
4. 生成 `buf generate`
